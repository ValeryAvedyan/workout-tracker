План Тестирования

1. Цели тестирования

1.1. Функциональное тестирование: Проверить, что все функции (добавить, удалить, сохранить, статистика) работают правильно.

1.2. Пользовательское тестирование: Оценить удобство интерфейса для обычных пользователей.


2. Объекты и инструменты

2.1. Объекты: Главные классы Model, Controller, View и FileHandler.

2.2. Среда: IntelliJ IDEA, Java 11+, ОС Windows/macOS.

2.3. Инструменты: Ручная проверка, опрос через Google Forms.


3. Успешное завершение

3.1. Функционал: Приложение работает без критических ошибок.

3.2. Обратная связь: Есть отзывы минимум от 3 пользователей, они проанализированы.
