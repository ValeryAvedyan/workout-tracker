Workout Tracker - система учёта физических тренировок

1. Название проекта
Workout Tracker - программный модуль для записи и анализа силовых тренировок

2. Описание проекта
Программный модуль на Java с графическим интерфейсом для систематического учёта физических тренировок. Предназначен для спортсменов и любителей фитнеса, которые хотят отслеживать свой прогресс в силовых упражнениях.

Основные функции:
- Запись тренировок по 12 видам упражнений
- Просмотр полной истории тренировок
- Статистика по топ-3 рекордам для каждого упражнения
- Сохранение и загрузка данных в CSV-формате
- Удобный графический интерфейс с вкладками

3. Установка и запуск

Установка Java FX:
1. Скачайте Java FX SDK с официального сайта
2. Выберите версию для вашей операционной системы
3. Распакуйте архив в удобное место (например, C:/javafx-sdk-17)

Настройка проекта в IntelliJ IDEA:
1. Скачайте проект с GitHub
2. Откройте проект в IntelliJ IDEA
3. Настройте Java FX:
   - File > Project Structure > Libraries > "+" > Java
   - Укажите путь к папке lib из распакованного Java FX
   - Apply > OK
4. Настройте конфигурацию запуска:
   - Run > Edit Configurations > Application
   - Main class: controller.MainApp
   - VM options: --module-path "путь/к/javafx-sdk/lib" --add-modules javafx.controls,javafx.fxml
5. Запустите приложение:
   - Откройте файл MainApp.java в папке controller
   - Нажмите Run 'MainApp.main()'

4. Примеры использования

Добавление тренировки:
1. Запустите приложение
2. Перейдите во вкладку "Добавить"
3. Выберите упражнение из списка (12 вариантов)
4. Введите данные: подходы, повторения, вес
5. Нажмите "Добавить"

Просмотр статистики:
- Перейдите во вкладку "Статистика"
- Нажмите "Обновить статистику" для просмотра топ-3 рекордов

Управление данными:
- Сохранение данных: кнопка "Сохранить" во вкладке "Тренировки"
- Загрузка данных: кнопка "Загрузить" во вкладке "Тренировки"
- Удаление записи: выделите запись в таблице и нажмите "Удалить"

Скриншоты интерфейса находятся в репозитории и демонстрируют работу приложения.

5. Структура репозитория

Workout-Tracker/

- src/controller/ - Управление логикой приложения
  - MainApp.java - Главный класс для запуска
  - WorkoutController.java - Основной контроллер тренировок
- src/model/ - Модели данных
  - FitnessModel.java - Основная бизнес-логика
  - WorkoutRecord.java - Класс записи о тренировке
- src/view/ - Пользовательский интерфейс
  - MainFrame.java - Главное окно приложения
  - WorkoutsPanel.java - Панель просмотра тренировок
  - AddWorkoutPanel.java - Панель добавления тренировок
  - StatisticsPanel.java - Панель статистики
- src/util/ - Вспомогательные классы
  - FileHandler.java - Работа с файлами
- src/data/ - Хранение данных
  - workouts.csv - Файл с данными тренировок
- screenshots/ - Скриншоты интерфейса приложения
- README.md - Документация

6. Технические требования

- Язык программирования: Java 11 или выше
- Графическая библиотека: JavaFX 17 или выше (требует отдельной установки)
- Интегрированная среда: IntelliJ IDEA
- Операционные системы: Windows, macOS, Linux
- Минимальные требования: 2 GB RAM, 500 MB свободного места
